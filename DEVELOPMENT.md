# BUILDTIME REQUIREMENTS

# REQUIREMENTS

* [Node.js](https://nodejs.org/en/) 20.17.0+ with `corepack enable`
* [Snyk](https://snyk.io/)
* [Yarn](https://yarnpkg.com/) 4.5.0+

## Recommended

* [ASDF](https://asdf-vm.com/) 0.14.1 (run `asdf reshim` after provisioning)

# AUDIT: NPM AUDIT + SNYK

```console
$ npm run audit
```

## NPM AUDIT

```console
$ npm audit
```

## SNYK

```console
$ npm run snyk
```
