# INSTALL

We support multiple installation methods.

## REQUIREMENTS

* [Node.js](https://nodejs.org/en) 20.17.0+

## NPM

```sh
npm install @mcandre/periscope --save-dev
```

## Build from Source

```sh
npm install [-g]
```

For more information on developing periscope itself, see [DEVELOPMENT.md](DEVELOPMENT.md).
